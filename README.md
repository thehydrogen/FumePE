# Fume PacketEvents (FPE)
Fume PacketEvents is a fork of [PacketEvents](https://github.com/retrooper/packetevents).

## About
FPE was originally created to have a version of PacketEvents which is tested to be stable and has some other extra features (such as distinguished Client/Server packet types).
